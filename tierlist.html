Tier List Creator

Project Summary

I worked on an application in Java that lets users organize items into tier lists based on different categories. Users can either choose from existing categories, create their own custom tier lists, or generate random ones. Users can also create accounts to save their tier lists and access them later. To add more personalization, we used the ChatGPT OpenAI API to help users create tournaments by generating lists of relevant items.

The primary focus of this project was not to create a complex application but instead to rigidly adhere to the SOLID principles and the concept of Clean Arcitecture (CA), as well as implementign design patterns that we had learnt in class such as adapter and builder. Our code was meticulously organised into sections to follow CA as follow: Interface Adapters and View, Application Business Rules and DAO and Entities.

When it comes to my contribution with this project, I coded essentially all of the front end of the application, as well as integrating the front and backend, so this was great practice when it came to UI/UX design.
