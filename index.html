DICOM Image Conversion Python

I developed a Python pipeline to convert 2D Cartesian images into 3D DICOM files, enhancing their use in medical imaging. I implemented spatial alignment and volumetric reconstruction algorithms for accurate 3D models, optimizing the process for large datasets while ensuring data integrity. I validated the 3D DICOM files using viewers like Weasis, 3D Slicer, and OsiriX.

Diagnosis Classification Python

I worked on a project where I transformed diagnosis text files into structured JSON, making data easier to manage and store. I also developed a Python interface that allowed healthcare professionals to label conditions across more than half a million diagnosis documents. To support AI model training, I created a large dataset from unstructured text for extracting and labeling diagnoses.

Assembly Tetris Assembly

I completed a project where I used MIPS assembly to create a version of the classic Tetris game. Since we didn't have access to physical MIPS hardware, I simulated the game using MARS. This simulator allowed me to not only work with the MIPS processor but also to simulate a Bitmap Display and Keyboard input, bringing the game to life in a virtual environment.
